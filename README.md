# ProstoLab

Next.js digital production agency website with advanced scroll animations.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started

### Prerequisites
- Node.js 18+ installed

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Features
- **Pinned Hero Section**: Scroll-driven animation sequence.
- **3D Tag Cloud**: Deterministic positioning with perspective transforms.
- **Responsive**: Adapts tag count and layout for mobile.
- **Accessible**: Semantic HTML and keyboard navigation support.

## Testing
Run smoke tests with Playwright:
```bash
npx playwright test
```
